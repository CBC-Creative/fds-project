Viewpoint’s Proprietary Sensor Sharing Technology

Sensor Sharing exists at the ‘Area’ level. In order for multiple units to share sensors, they need to be set up to the same Area. Once two or more units are set up to the same Area, Sensor Sharing will happen automatically. Array is designed such that all units in an Area will respond to the ‘Worst Case’ of any unit within that Area. If one unit sees a contaminant, all units in that Area will respond to the contaminant. All units will remain in a heightened fan state until the original triggered sensor has returned to normal. Once that happens, the system will check that all other sensors in the Area have returned to normal before allowing all units to return to a lower fan speed.
